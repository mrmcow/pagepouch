## Screenshot Post-Processing

### Tools Needed:
- **Mac**: Preview (built-in) or https://www.photopea.com (free online)
- **Windows**: Paint, Paint 3D, or https://www.photopea.com
- **Any OS**: Figma (free), Canva (free tier)

### Steps:

1. **Resize to exactly 1280x800px**
   ```
   - Open image in editor
   - Resize canvas to 1280x800
   - Center content if needed
   ```

2. **Add subtle border (optional but recommended)**
   ```
   - Add 1-2px border in light gray (#E5E7EB)
   - Makes screenshots pop against white background
   ```

3. **Compress for file size**
   ```
   - Use TinyPNG.com or similar
   - Keep under 5MB per image (stores requirement)
   - Aim for 500KB-2MB per screenshot
   ```

4. **Add annotations (optional)**
   ```
   - Use arrows to highlight key features
   - Add callout boxes for important elements
   - Keep annotations minimal and professional
   ```

5. **Name files clearly**
   ```
   01-extension-popup.png
   02-dashboard-view.png
   03-search-results.png
   04-clip-detail.png
   05-folder-organization.png
   06-knowledge-graph.png
   07-mobile-responsive.png
   08-settings-profile.png
   ```

## ğŸ¯ Design Tips

### Color Palette:
- Primary: #3B82F6 (Blue)
- Secondary: #6366F1 (Indigo)
- Accent: #8B5CF6 (Purple)
- Text: #1F2937 (Dark Gray)
- Background: #FFFFFF (White) or gradient

### Typography:
- Headings: Inter, SF Pro, or similar (bold)
- Body: Inter, SF Pro, or similar (regular)
- Keep font sizes large enough to read at small sizes

### General Guidelines:
1. **Keep it simple** - Don't overcrowd
2. **High contrast** - Easy to read at any size
3. **Professional** - No clipart or low-quality images
4. **Consistent branding** - Use same colors/fonts as website
5. **Show real product** - Use actual screenshots, not mockups
6. **Readable text** - Even when scaled down
