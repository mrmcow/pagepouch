## üí° PRO TIPS

**Save Time:**
- Create Screenshot 1 as a template
- Duplicate and modify for 2-4
- Use Canva's "Copy Style" feature
- Work in batches (all backgrounds, then all text)

**Ensure Quality:**
- Use real app data (not mockups)
- Check contrast (text must be readable)
- Add subtle shadows (instant polish)
- Keep it simple (less is more)
- Review on mobile preview

**Avoid Mistakes:**
- Don't use too many colors (stick to brand)
- Don't cram too much text
- Don't use blurry screenshots
- Don't skip the checklist
- Don't forget PRO badge on #5
