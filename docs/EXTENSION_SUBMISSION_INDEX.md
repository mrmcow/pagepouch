## ‚ö° Quick Reference

### What You Need to Create:
1. **5-8 Screenshots** (1280x800px) - See Assets Guide
2. **1 Promo Image** (440x280px, Chrome only) - See Assets Guide
3. **Test Account** - Create at pagestash.app/auth/signup
4. **Developer Accounts** - Chrome ($5), Firefox (free)

### Time Estimates:
- Screenshots: 1-2 hours
- Promo images: 30-60 minutes
- Account registration: 15 minutes
- Chrome submission: 30 minutes
- Firefox submission: 30 minutes
- **Total**: 3-5 hours active work

### Costs:
- Chrome Web Store: $5 one-time
- Firefox Add-ons: Free
- Design tools: Free (Canva/Figma free tiers)
- **Total**: $5

### Review Times:
- Chrome: 1-7 days (sometimes longer)
- Firefox: 30 minutes - 14 days
