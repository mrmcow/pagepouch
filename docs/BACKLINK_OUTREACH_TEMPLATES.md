## üí° **Pro Tips:**

1. **Personalize the first line** - Reference something specific from their site
2. **One email = One ask** - Don't combine multiple requests
3. **Provide value first** - What's in it for them?
4. **Keep it scannable** - Short paragraphs, bullet points
5. **Make it easy** - Provide links, details, everything they need
6. **Follow up ONCE** - After 7 days, only if no response
7. **Track opens** - Use tool like Streak to see if they opened
8. **Best times** - Tuesday-Thursday, 10am-2pm in their timezone
