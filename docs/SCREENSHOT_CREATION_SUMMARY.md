## üÜò STUCK? QUICK SOLUTIONS

### **Problem: Don't know Canva**
**Solution:** 
- Watch 10-min YouTube: "Canva tutorial for beginners"
- Use pre-made templates (search "social media post")
- Copy exactly from guide, one step at a time

### **Problem: Screenshots look ugly**
**Solution:**
- Use more shadows (makes everything better)
- Increase spacing (more whitespace)
- Stick to brand colors (don't add random colors)
- Keep it simple (less is more)

### **Problem: Taking too long**
**Solution:**
- Use "Side by Side" layout for all (simplest)
- Don't overthink it (good enough > perfect)
- Duplicate first screenshot as template
- Skip optional flourishes

### **Problem: Text not readable**
**Solution:**
- Make text larger (18px minimum)
- Use darker color (#0F172A, not gray)
- Add background panel for text
- Check contrast (dark text on light background)
