**Primary Font:** [Inter](https://fonts.google.com/specimen/Inter)

- Fallback: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif

## üé® KNOWLEDGE GRAPH SCREENSHOT - VISUAL GUIDE

### **What to Show in Your Graph:**

**Nodes (circles/bubbles):**
- 10-15 visible nodes minimum
- Different sizes (hub nodes = larger)
- Color-coded by type:
  - Blue (#2563EB): Clip nodes
  - Purple (#A855F7): Folder/topic clusters  
  - Green (#10B981): Domain hubs
  - Gray (#94A3B8): Supporting entities

**Connections (lines between nodes):**
- Multiple connections visible
- Varying thickness (stronger connections = thicker)
- Different colors:
  - Blue: Same source
  - Purple: Same topic
  - Green: Content similarity
  - Orange: Temporal proximity

**Labels:**
- At least 5-8 node labels visible
- Short, readable text (e.g., "TechCrunch", "AI Research", "Nov 2024")
- Not cluttered - some nodes can be unlabeled

**Layout:**
- Clustered groups visible (related nodes close together)
- Clear center/focal point
- Some nodes on periphery
- Organic, not grid-like

### **How to Create the Visual:**

**Option 1: Screenshot from PageStash**
1. Go to Knowledge Graph in your dashboard
2. Create a graph with 20-30 clips
3. Zoom to show good overview
4. Screenshot the canvas

**Option 2: Mock in Figma/Canva**
1. Create circles of various sizes
2. Add connecting lines
3. Color-code nodes
4. Add short labels
5. Group into visible clusters

**Option 3: Use a Network Graph Template**
1. Search Canva for "network diagram"
2. Customize with your brand colors
3. Add realistic labels
4. Adjust node sizes
5. Make it look organic, not too perfect

### **Pro Tips for Knowledge Graph Screenshots:**

‚úÖ **Do:**
- Show clear clustering (groups of related nodes)
- Use varying node sizes (visual hierarchy)
- Make some connections thicker than others
- Label key nodes (domains, topics)
- Show at least 3-4 distinct clusters
- Use brand colors (#2563EB, #A855F7)
- Add subtle glow/shadow to nodes

‚ùå **Don't:**
- Make it too cluttered (15 nodes max visible)
- Use too many colors (stick to 3-4)
- Make all nodes the same size
- Label every single node
- Make connections cross too much
- Use random/meaningless labels
- Make it look like a corporate org chart

### **Recommended Tools:**

1. **PageStash itself** (most authentic)
2. **Figma** (network graph plugins available)
3. **Canva** (search "network" or "mind map")
4. **NetworkX visualization** (if you code)
5. **Miro** (has graph templates)
