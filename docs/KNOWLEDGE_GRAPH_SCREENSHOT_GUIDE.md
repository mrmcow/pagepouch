## ðŸŽ¨ NODE COLORS & TYPES

- **Blue (#2563EB)**: Clip nodes (main content)

- **Large (60-80px)**: Hub nodes (5+ connections)
- **Medium (40-50px)**: Regular clips (2-4 connections)

- **Blue (#3B82F6)**: Same source connections

### **Connection Thickness:**
- **Thick (3-4px)**: Strong relationship
- **Medium (2px)**: Moderate relationship
- **Thin (1px)**: Weak relationship

## ðŸ› ï¸ HOW TO CREATE IN CANVA

### **Step 1: Background**
```
1. Create 1280 x 800px canvas
2. Background: 
   - White to light blue gradient
   - OR: Solid white
3. Add subtle grid (optional)
```

### **Step 2: Create Nodes**
```
1. Insert circles:
   - Small: 30px
   - Medium: 50px  
   - Large: 70px
   
2. Colors:
   - Fill with brand colors
   - Add subtle shadow
   - Optional: inner glow
   
3. Position:
   - Create 3-4 visual clusters
   - Leave space for connections
   - Organic placement (not grid)
```

### **Step 3: Add Connections**
```
1. Use line tool
2. Draw between related nodes
3. Vary thickness (1-4px)
4. Use different colors
5. Avoid too many crossings
```

### **Step 4: Add Labels**
```
1. Text: Inter, 12-14px, Semi-Bold
2. Position: Inside OR next to nodes
3. Color: White (for colored nodes) or #0F172A
4. Only label 5-8 key nodes
```

### **Step 5: Add Copy Panel**
```
1. Create rounded rectangle
   - Width: 480px
   - Height: ~550px
   - Corners: 24px
   
2. Fill: White with 95% opacity
3. Shadow: 0 10px 30px rgba(0,0,0,0.15)
4. Position: Left or right side
5. Padding: 48px all sides
```

### **Step 6: Add Text**
```
1. Headline: Inter Bold 56px, gradient
2. Subheading: Inter Semi-Bold 28px
3. Features: Inter Regular 20px, with bullets
4. Badge: Gradient background, white text
5. Stats: Small text at bottom
```
